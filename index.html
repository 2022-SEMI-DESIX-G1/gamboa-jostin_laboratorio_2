¿Qué es javaScript? ¿En qué se asemeja a Java?

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

La semenjansa que tinene javaScrip con java es que son orientados a objetos y los dos son lenguajes interpretados, con la diferencia que java es interpretado y luego compilado.
